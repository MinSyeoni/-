*주의*
툴 끌 때 나타나는 에러 메시지 무시해도 됨.

툴 경로 SYTool/Tool/Debug/Tool.exe 실행

1. 툴 이동 방법
W,A,S,D 화면 상하좌우 이동
마우스 우클릭 한 상태로 키 누르면 그 방향으로 회전

2. 먼저 Terrain 띄워야 함. Terrain 툴 사용 법
x 129, y 129, z 1로 고정되있음, create 버튼 누를 것.
Solid 모드는 텍스쳐로 채워진 버전, Wire 모드는 Terrain 틀만 나타남. 

3. 오브젝트 툴 사용법
Collider 모드 버튼 선택 풀고 작업해야 함. 
Terrain 띄우고 마우스 왼쪽 클릭으로 지형 클릭하면 그 위치 정보가 x,y,z좌표에 나타남.

(1) create 모드
리스트에서 띄우고 싶은 오브젝트 선택 후 create하면 위 위치에 띄워짐.
*지형 말고도 오브젝트 위에 얹어서도 생성 가능*

(2) modify 모드
오브젝트 클릭하고 x,y,z position / x,y,z rotation / x,y,z scale 칸에서 마우스 휠로 수치 조정 가능.

(3) delete 모드
오브젝트 클릭하고 delete 버튼 누르면 바로 삭제 

(4) save 버튼
띄운 오브젝트 전부 저장. 파일 이름 저장할 것.

(5) load 버튼
저장했던 오브젝트 전부 불러옴.

4. Collider 툴 사용법
Collider 옵션에서 콜라이더 모드 선택하면 그 후 부턴 오브젝트 관련 행동 할 수 없음, 
충돌체 관련 작업만 할 수 있음.
지형 생성 / 오브젝트 생성 후 충돌체 생성 가능.

(1) Create 모드
지형 위나 오브젝트 위 마우스 왼쪽 클릭하면 위치 값 나타남.
Box 모양 / Sphere 모양 선택 가능. 
create 버튼 누르면 그 위치에서 충돌체 생성.
충돌체인지 트리거인지 다양한 옵션 줄 수 있음. 지금은 필요 X, 그냥 1번으로 놔둬도 됨.

(2),(3),(4),(5) 오브젝트 툴이랑 같음. 


5. NaviTool 사용법
*플레이어가 걸어갈 수 있는 땅 위에만 길을 만들어주면 됨*

(1) Create 모드
땅이나 오브젝트 위에 삼각형 모양으로 세번 클릭하면 삼각형이 입혀짐
플레이어는 삼각형이 그려진 땅위에서만 걸어갈 수 있음.
반시계 방향으로 그리면 안되고 시계방향으로 그려야 함
하나의 꼭지점에 겹쳐서 그릴 수 있음. 

(2) Modify모드
삼각형 꼭지점 클릭하면 빨간색으로 변함. 맵툴 처럼 꼭지점의 위치가 칸에 뜸. 마우스 휠로 조정 가능.

(3) Delete모드
Delete 버튼 누르면 *뒤에서부터* 하나씩 삼각형이 사라짐. 
많이 깐 후 중간에 잘못 만들었다면 지울 때 시간 많이 걸려서 못지움..
네비를 잘못 깔았다 싶으면 바로바로 지워줘야 함.

(4) Save, Load 다른 툴과 동일.

네비 리스트 몇 번 셀 클릭하면 삼각형 전체가 빨간색으로 변함. 
셀 찾고 싶을 때 이용. 근데 지금 갑자기 삭제했다가 다시 생성하면 리스트 추가가 안됨 ㅠ
